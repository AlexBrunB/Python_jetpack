# README #


### What is this repository for? ###

* This is a directory to learn python programming using cocos2d library
* requirement: python2.7

### Set up? ###

* First install python 2.7
* Then create a virtual env with mkvirtualenv python2.7
* In this virtual env install cocos2d:
    - pip install cocos2d --process-dependency-links
    - cd cocos2d-0.6.0
    - setup.py install
* Clone this repository by:
    - hg clone ssh://hg@bitbucket.org/alexbrun/python_jetpack


### Run? ###

- cd exemples
- python hello_alex.py
